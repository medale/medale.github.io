# Exploring Hardware Environment

## Determine 32 vs. 64-bit OS
    >file /sbin/init
    /sbin/init: ELF 64-bit LSB shared object, x86-64,...
    
## Determine Ubuntu Version and Version Name
    >lsb_release -a
    Description:	Ubuntu 13.10
    Release:	13.10
    Codename:	saucy
