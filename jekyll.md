# Install Jekyll via rvm
* Install Ruby version manager https://rvm.io/rvm/install
* curl -L https://get.rvm.io | bash -s stable
* source ~/.rvm/scripts/rvm
* gem install jekyll

# Run jekyll static site generation

     jekyll serve

# Updating jekyll via gems

     gem update jekyll
